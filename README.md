**Minimal Resume Studio**

A clean and minimal web application for creating resumes, building portfolio sections, and running ATS (Applicant Tracking System) keyword checks — all in one responsive interface.
Designed to work smoothly on laptops, tablets, phones, and even ultra-wide screens.

**Features:**

Resume Builder with live preview

Portfolio Builder with clean UI

ATS Score Checker based on keyword overlap

Light & Dark Theme Toggle

Fully Responsive design across all devices

Minimal, fast and distraction-free interface

Frontend only (for now) — backend planned with FastAPI
